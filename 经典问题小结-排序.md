## 冒泡排序：O(n^2)|O(1)

最坏时间复杂度O(n*(n-1)/2) = O(n^2)

空间复杂度O(1)

```
vector<int> sortArray(vector<int>& nums) {
        int flag;
        for(int i = 0;i < nums.size() - 1;i++){
            flag = 1;
            for(int j = 0; j < nums.size() - 1 - i;j++){
                if(nums[j] >= nums[j + 1]){
                    swap(nums[j],nums[j+1]);
                    flag = 0;
                }
            }
            if(flag) break;
        }
        return nums;
    }
```
